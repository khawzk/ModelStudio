Alibaba Cloud Model Studio

BYOK demo kit for customer-facing AI discovery.

A polished Model Studio console for Qwen, Wan, image editing, video, speech, vision, and realtime translation. Users paste their own DashScope API key at runtime.

View Demo Code GitHub Repository

Bring Your Own Key

No API key is stored in the repo or in GitHub Pages. The running demo accepts each user's key from the UI.

Customer Ready

Built for Alibaba Cloud Telesales MYS demo conversations, with focused workflows instead of a generic notebook surface.

Light Hosting

Run locally or on a temporary ECS. No Terraform is required; stop the host after the session to control cost.

Run locally

Start the live console

GitHub Pages is static. Live API calls use the Python app because realtime translation needs a backend WebSocket proxy.

git clone https://github.com/khawzk/ModelStudio.git
cd ModelStudio/AI_Model_Studio_Portal
python3 -m pip install -r requirements.txt
python3 server.py

Capabilities

Included demo modules

  • Text Intelligence with Qwen-compatible chat models
  • Vision Lab with image URL and upload support
  • Omni realtime WAV translation with LiveTranslate
  • Image Studio for edit, generation, fusion, and camera upload
  • Video Studio for async text-to-video and image-to-video
  • Speech AI for uploaded audio transcription
